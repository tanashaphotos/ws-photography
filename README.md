# Tanasha Photos

Landing page desarrollada para mostrar el portafolio fotográfico de Tania Espino, enfocada en una experiencia moderna, elegante y totalmente adaptable a dispositivos móviles.

La página fue diseñada con el objetivo de transmitir una estética visual limpia y profesional, dando prioridad a las fotografías, la navegación fluida y la presencia digital de la fotógrafa.

---

## Características

- Diseño moderno y minimalista.
- Experiencia responsive para móvil, tablet y escritorio.
- Optimización visual para portafolio fotográfico.
- Integración con redes sociales.
- Transiciones y animaciones suaves.
- Deploy en GitHub Pages.

---

## Tecnologías utilizadas

- Next.js.
- React.
- Tailwind CSS.
- TypeScript.

---

## Proyecto

Este sitio funciona como una plataforma visual para presentar el trabajo fotográfico de manera rápida, moderna y accesible.
